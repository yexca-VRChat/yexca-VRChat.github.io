    Animation

      • 开关
      • 动作
        • 现成的
        • 自行制作
      • 其他

    首先是制作动画，动画的制作一般是一个物品开关

    开关 #

    物品开关即录制动画把一个物品打开 同时这个物品可以放在自己的身体部位上（物体绑骨），例如手部

    粒子特效一般也可以通过录制开关进行播放

    动作 #

    动作动画一般是MMD的舞蹈动作或Pose动作，建议使用现成的，自己制作较为困难

    现成的 #

    对于MMD舞蹈，一般可以使用相应的工具进行转换为Unity可以识别的文件，对于Pose动作一般有成品可以进行购买使用

    自行制作 #

    如果使用Unity进行制作是不现实的（动作可以在动画窗口添加属性-Animator里找），可以在其他软件制作骨骼动画以导入Unity

    其他 #

    例如物品移动，表情动画，Shader改变以改头发颜色等
